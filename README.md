# Libreria

### Descripción
Permite crear, ediar , filtrar por id, listar y eliminar libros y autores

### Stack
- NodeJS
- Express
- Sqlite
- Prisma
- Typescript

### Instalación
1. Clonar el repositorio
2. Instalar dependencias
```bash
npm install
```
3. Correr el proyecto
```bash
npm run dev
```
4. Correr la seed
```bash
npm run seed
```
